## Database

Opções de validação:
- NOT NULL
  - `"name" TEXT NOT NULL`
- UNIQUE
  - `CREATE UNIQUE INDEX "User_email_key" ON "User"("email");`
- DEFAULT
  - `"createdAt" DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,`
- CHECK
  - `"password" TEXT NOT NULL CHECK(LENGTH(password) >= 8)`
